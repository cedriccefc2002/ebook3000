# ebook3000

ebook3000 網站有太多廣告，所以寫了資料一個工具直接抓取下載網址

## 編譯

```sh
npm i
npx tsc
```

## 執行方式

```sh
node . > out.md
```

產生 out.md 圖片與下載網址

使用VS CODE 開啟 out.md
